## Overview

File management efficiency scales directly with the tools available for it. <a href="#">TotalFinder</a> provides a set of Finder enhancements that compound in practical value — tabs, dual-pane, cut-and-paste, and visual organisation working together create a file management experience that handles complex workflows that stock <a href="#">Finder</a> cannot accommodate without significant friction.

The <a href="#">cut and paste</a> operation for files — Command-X to cut, Command-V to paste at destination — brings keyboard-initiated file moving to Finder. Stock Finder requires drag-and-drop for every move operation, which is impractical when source and destination are in different parts of the directory hierarchy that cannot both be visible simultaneously. Cut and paste makes move operations as keyboard-accessible as copy operations, resolving a limitation that users coming from other operating systems notice immediately. <a href="#">Keyboard shortcuts</a> for tab creation, switching, and management match the efficiency of browser tab workflows, making the enhanced Finder fast to navigate without requiring mouse interaction for every operation.

The <a href="#">system folder visibility</a> toggle makes hidden Unix directories including /Library, /etc, and /usr accessible from the Finder sidebar for users who need to navigate these locations graphically. For developers and administrators who regularly access system directories, this is a meaningful convenience over Terminal navigation for operations that involve Quick Look, drag-and-drop, or other Finder capabilities. <a href="#">Session persistence</a> of all open tabs means a working context of several locations — project folders, archive locations, frequently used directories — is restored automatically each time Finder opens, eliminating the overhead of reconstructing the working context from scratch each session. TotalFinder's combination of <a href="#">tab management</a>, dual-pane layout, cut-and-paste, and visual organisation transforms Finder into a file management environment that serves power user requirements without requiring a different application.

## Key Features

- Browser-style <a href="#">tabs</a> for multiple folder locations in one window
- <a href="#">Dual-pane</a> side-by-side layout for two folder locations simultaneously
- File <a href="#">cut and paste</a> with Command-X and Command-V support
- Coloured <a href="#">folder labels</a> as full icons in the sidebar
- <a href="#">Visor mode</a> sliding Finder from screen edge via keyboard shortcut
- <a href="#">Folders on top</a> sorting keeping directories above files
- <a href="#">System folder</a> visibility toggle for hidden directory access
- Session-persistent <a href="#">tab pinning</a> for always-accessible locations
- All native <a href="#">Finder integration</a> preserved including iCloud and AirDrop

## Additional Information

Tab persistence across sessions reflects how file management contexts actually work. Users maintaining specific working contexts benefit from locations being restored automatically rather than requiring reconstruction each time Finder opens — making TotalFinder's tab system a persistent productivity improvement rather than a temporary convenience.

Coloured label visibility as full-colour folder icons rather than small dots makes visual organisation effective at sidebar scale. Full-colour icons create a readable visual hierarchy at a glance across the complete sidebar list without concentrated attention on each small indicator.

Visor mode is particularly valuable for full-screen application users. Accessing the file system from a full-screen writing environment or code editor without switching out of the full-screen context keeps the primary workflow intact while providing file system access on demand via keyboard shortcut.
